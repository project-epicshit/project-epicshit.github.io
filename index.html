NetApp in a Cloud DevOps Context!

on February 8, 2022

﻿NetApp in a Cloud DevOps Context! Are you running a Kubernetes platform in the cloud based on cloud provider native services today? Then let me show you how NetApp can help to optimize this environment. Each cloud provider uses its own storage products for its Kubernetes services, which are addressed via the CSI driver. In a NetApp optimized world, the “standard” driver will be replaced by Astra Trident. With Trident, it is now possible to use PVC on a cloud-optimized ONTAP: Cloud Volume Service, Azure NetApp Files, Cloud Volume ONTAP or FSx for NetApp ONTAP are the perfect solutions.
